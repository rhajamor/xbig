The XBiG XML Schema(s)

XBiG Home

This is the home of the XBiG 1.0 XML Schemas from the XBiG Sourceforge Development Team.

This schema is not normative. It is only provided as a tool to validate XBiG instances, without any guarantee on the accuracy of the results. This version will be updated gradually to incorporate corrections or changes.

The schema files are available for download as a tgz archive, or can be browsed on line.
